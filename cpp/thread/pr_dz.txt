Задачи для практики с многопоточностью в C++
Задача 1: Потоки и суммирование массива
Цель: Написать программу, которая разделяет массив чисел на части и использует множество потоков для параллельного суммирования элементов каждой части. В конце основной поток должен собрать и вывести сумму всех частей.

Детали:

Разделите массив на N равных частей (N — количество потоков).
Каждый поток суммирует свою часть массива.
В конце, основной поток суммирует все промежуточные суммы.



Задача 2: Многопоточный таймер
Цель: Создать многопоточное приложение с таймером, где каждый поток начинает отсчёт заданного времени (разное для каждого потока), выводит уведомление при достижении этого времени и завершает свою работу.

Детали:

Каждый поток получает уникальное количество секунд для отсчёта.
Потоки должны выводить сообщение каждую секунду о том, сколько времени осталось.
По окончании времени каждый поток должен вывести сообщение о завершении работы.




Задача 3: Многопоточный поиск простых чисел
Цель: Разработать программу, в которой каждый поток проверяет числа из заданного диапазона на простоту и выводит найденные простые числа.

Детали:

Разделите диапазон чисел между несколькими потоками.
Каждый поток проверяет числа в своём диапазоне и выводит простые числа.
В конце основной поток может, например, вывести общее количество найденных простых чисел или просто завершить программу.
Эти задачи помогут укрепить понимание работы с многопоточностью, синхронизацией и использованием различных потоковых операций в C++.
